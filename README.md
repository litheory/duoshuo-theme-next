## 用法

- `\next\source\css\_common\components\third-party`,找到`duoshuo.styl`文件，这个就是控制**多说评论版块样式**的css文件，可以直接替换。
- 或者直接在**多说官网**的开发者中心中添加**css**，但并不推荐这种方式，因为在多说官网上修改的css很容易被本地的`.styl`覆盖。
